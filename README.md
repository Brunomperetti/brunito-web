# brunito-web

Landing web creada desde cero con HTML, CSS y JavaScript puro, sin frameworks ni librerías externas.

## Estructura

```text
.
├── index.html
├── css/
│   └── styles.css
├── js/
│   └── main.js
├── images/
├── README.md
└── .gitignore
```

## Secciones iniciales

- Header
- Hero
- Servicios
- Cómo trabajo
- Experiencia / casos
- Sobre brunito
- Contacto
- Footer

## Cómo verla localmente

Abrí `index.html` directamente en el navegador o levantá un servidor local simple, por ejemplo:

```bash
python3 -m http.server 8000
```

Después ingresá a `http://localhost:8000`.
